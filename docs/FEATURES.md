# Yue AI 助手功能特性概览 (Features Overview)

本文档汇总了 Yue AI 助手目前已实现的核心功能与交互特性，旨在提供高效、直观的人机协作体验。

## 1. 核心交互 (Core Interaction)

### 聊天名称标识 (Teams Style Sender Header)
- **角色名显示**: 在每条消息气泡上方展示发送者名称，用户为 "You"，AI 为当前智能体名称。
- **对话清晰度**: 参考 Teams 风格强化角色归属感，提升多智能体协作场景的可读性。

### 思维链可视化 (Thinking Process Visualization)
AI 的深度思考过程不再是黑盒，我们参考了 DeepSeek R1 与 Gemini 的设计理念，实现了透明化的思维展示：
- **可折叠面板**: 思考过程被封装在独立的灰色区域中，默认折叠，保持对话流整洁。
- **实时计时**: 标题栏实时显示 "Thinking Process" 及耗时（秒/分），让等待过程有据可循。
- **内容解析**: 自动解析模型输出的 `<thought>` 或 `<think>` 标签。
- **视觉优化**: 采用 Full-bleed（全宽）设计与等宽字体 (Monospace)，提供代码级的阅读体验。

### 代码与预览工坊 (Artifacts & Preview)
参考 Claude Artifacts 的先进体验，我们将代码生成从单纯的文本展示升级为可视化交互：
- **实时渲染**: 当 AI 生成 HTML、SVG 或 React 代码时，右侧 "Intelligence Hub" 面板会自动激活预览模式。
- **支持类型**:
  - **SVG**: 直接渲染矢量图形，支持缩放查看。
  - **HTML/Tailwind**: 实时预览网页布局与样式。
  - **React**: 渲染交互式组件。
- **智能检测**: 系统自动识别代码块语言，并在代码块右上角提供 "Preview" 入口（如果未自动打开）。
- **可扩展分屏 (Expandable Split View)**:
  - 默认提供 **420px** 的舒适阅读宽度。
  - 支持 **一键宽屏** (Expand Mode)，将预览区域扩展至屏幕宽度的 **55%**，为复杂网页、图表提供沉浸式查看体验。

## 2. 消息操作 (Message Actions)

为了提升内容的可用性与流转效率，我们重新设计了消息操作区域：

### AI 回复操作栏 (Assistant Actions)
位于每条 AI 回复的**底部**，图标**始终可见**，无需悬停即可操作：
- **📋 复制 (Copy)**: 一键复制 Markdown 格式的完整回复内容。
- **🔄 重新生成 (Regenerate)**: 对不满意的回复进行重新生成。
- **🔊 朗读 (Read Aloud)**: *(UI Ready)* 文本转语音朗读入口。
- **⬇️ 下载 (Download)**: *(UI Ready)* 将回复保存为文件。
- **share 分享 (Share)**: *(UI Ready)* 生成分享链接。
- **... 更多 (More)**: 更多扩展功能入口。

### 用户消息操作 (User Actions)
- **复制按钮**: 位于用户消息气泡左侧，始终显示，方便快速复用历史提示词 (Prompt)。

## 3. 界面体验 (UI/UX)

- **现代化视觉**:
  - 采用 **Glassmorphism (毛玻璃)** 效果，界面通透且富有质感。
  - **Tooltip 升级**: 使用深色半透明背景 (`bg-slate-900/95`) 配合模糊效果，提供专业且清晰的提示信息。
- **交互细节**:
  - 组件尺寸与间距经过精心调优，操作更加顺手。
  - 关键操作（如 Thinking Process 展开）增加了平滑的过渡动画。
- **响应式布局**:
  - 完美适配桌面端与移动端，侧边栏支持动态收起/展开。

## 4. 智能体与模型 (Agents & Models)

- **多模型切换**: 支持动态切换底层 LLM 模型（如 GPT-4, Claude 3.5, DeepSeek 等）。
- **智能体 @提及**: 在输入框输入 `@` 即可唤起智能体选择菜单，快速调用特定领域的 AI 专家。
- **多模态交互 (Multimodal)**:
  - **图片上传**: 支持在聊天中发送图片（PNG/JPG/GIF），AI 可识别并基于图片内容进行回答（需模型支持 Vision）。
  - **持久化存储**: 图片自动保存至本地，历史记录刷新不丢失。
- **文档检索子 Agent (Docs Agent)**:
  - **目录级检索**: 支持在用户指定的本地目录下检索 Markdown 文档。
  - **安全沙箱**: 具备严格的目录白名单与黑名单策略，防止敏感路径访问。
  - **结构化引用**: 回答自动附带 `citations`，支持展示文件路径与证据片段。
