# Historical Board of Directors - Routing Registry

This file is the routing table for the skill. Use it to decide:
- who should be the primary mentor
- who should serve as supporting voices
- where a persona fits well
- where a persona should be used cautiously

Choose based on the user's real dilemma, not on surface keywords alone.

## Routing Rules

- First identify the user's underlying tension, not just the topic.
- Prefer one primary mentor whose method best matches the core tension.
- Add two supporting voices that either widen the frame, correct bias, or add a complementary operating logic.
- Check `Caution Flags` before selecting a primary mentor.
- If the user explicitly names a figure, use that figure as the primary mentor unless the request clearly falls into that figure's caution zone. If so, honor the request but explicitly note the fit risk.
